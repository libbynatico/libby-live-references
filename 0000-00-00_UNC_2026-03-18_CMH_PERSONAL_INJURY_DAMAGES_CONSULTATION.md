**3. LEGAL FRAMEWORK**

**3.1 Duty of Care --- Negligence**

CMH owed Matthew Herbert a duty of care as a patient. That duty
included: correct diagnosis, adequate treatment, safe discharge
planning, and appropriate transfer when the hospital could not meet the
patient\'s needs.

+----------------------------------------------------------------------+
| **Legal source: Negligence --- common law duty of care**             |
|                                                                      |
| Ter Neuzen v. Korn, \[1995\] 3 SCR 674 --- Supreme Court of Canada   |
|                                                                      |
| Applied through Ontario civil courts.                                |
|                                                                      |
| Standard of care is judged against what a reasonable hospital in     |
| similar circumstances would have done.                               |
+----------------------------------------------------------------------+

**3.2 Thin Skull Doctrine --- Pre-Existing Conditions Do NOT Reduce the
Claim**

CMH will argue that Matthew\'s pre-existing Crohn\'s disease, ileostomy,
and TBI reduce their liability. This argument fails under the Thin Skull
Doctrine.

+----------------------------------------------------------------------+
| **Thin Skull Doctrine --- key principle:**                           |
|                                                                      |
| A defendant must take the plaintiff as they find them.               |
|                                                                      |
| If CMH\'s negligence caused or worsened harm to a vulnerable         |
| patient, CMH is fully liable for that harm --- regardless of         |
| pre-existing conditions.                                             |
|                                                                      |
| Source: Athey v. Leonati, \[1996\] 3 SCR 458 --- Supreme Court of    |
| Canada --- laws-lois.justice.gc.ca                                   |
|                                                                      |
| Crumbling Skull variant:                                             |
|                                                                      |
| Even if Matthew\'s condition would have deteriorated anyway, CMH     |
| accelerated or worsened that deterioration.                          |
|                                                                      |
| CMH is liable for the portion of harm they caused --- not the entire |
| pre-existing condition.                                              |
+----------------------------------------------------------------------+

**3.3 Unsafe Discharge --- Cascade Liability**

CMH discharged Matthew Herbert while unstable. At the time of discharge
Matthew was:

-   Homeless --- no fixed address

-   Unsupported --- no ODSP worker, no care coordinator

-   Post-surgical --- active ostomy complications

-   Cognitively impaired --- documented TBI

+----------------------------------------------------------------------+
| **Causation standard --- But For Test:**                             |
|                                                                      |
| But for CMH\'s negligent discharge, the downstream harms (further    |
| surgical complications, inability to manage ODSP, continued          |
| homelessness) would not have occurred or would have been reduced.    |
|                                                                      |
| Source: Clements v. Clements, 2012 SCC 32 --- Supreme Court of       |
| Canada --- laws-lois.justice.gc.ca                                   |

**4.3 Loss of Earning Capacity**

Matthew was homeless and unsupported at time of admission. The argument
is not lost wages --- it is lost earning capacity.

+----------------------------------------------------------------------+
| **Key distinction for this case:**                                   |
|                                                                      |
| Matthew was not working at time of admission. The argument is not    |
| lost wages.                                                          |
|                                                                      |
| The argument is: CMH\'s negligence compounded Matthew\'s existing    |
| disabilities and reduced his future capacity to ever earn income.    |
|                                                                      |
| Ontario courts have awarded \$100,000 --- \$400,000+ for loss of     |
| earning capacity where a young person\'s disabilities were worsened  |
| by negligence.                                                       |
|                                                                      |
| Source: Athey v. Leonati, \[1996\] 3 SCR 458 --- thin skull /        |
| crumbling skull applied to earning capacity.                         |

**4.5 Aggravated Damages**

Applies when the defendant\'s conduct was dismissive, high-handed, or
made the harm worse.

-   Denied transfer requests --- documented

-   Ignored or dismissed complaints --- documented

-   Patient relations complaints filed --- documented

+----------------------------------------------------------------------+
| **Source: Whiten v. Pilot Insurance Co., 2002 SCC 18 --- Supreme     |
| Court of Canada**                                                    |
|                                                                      |
| Aggravated damages apply when conduct was egregious and caused       |
| additional harm beyond the primary injury.                           |
|                                                                      |
| Range: \$25,000 --- \$100,000 additional on top of other categories. |
|                                                                      |
| Requires the complaints and dismissals to be documented in writing   |
| --- which they are.                                                  |

**5. MAKE WHOLE --- THE CASCADE ARGUMENT**

This is the argument that ties CMH\'s negligence to the broader systemic
failure.

+----------------------------------------------------------------------+
| **The argument in plain language:**                                  |
|                                                                      |
| CMH discharged Matthew Herbert --- homeless, post-surgical, TBI, no  |
| supports --- into a situation where failure was inevitable.          |
|                                                                      |
| That discharge did not just cause physical harm.                     |
|                                                                      |
| It caused: inability to manage ODSP, further surgical complications  |
| requiring other hospitals, continued homelessness, inability to      |
| work, loss of benefits.                                              |
|                                                                      |
| CMH is not only liable for what happened inside their walls.         |
|                                                                      |
| They are liable for the reasonably foreseeable downstream harm       |
| caused by discharging a vulnerable patient into homelessness without |
| supports.                                                            |
|                                                                      |
| Legal source: Clements v. Clements, 2012 SCC 32 --- but for the      |
| discharge, downstream harms would not have occurred.                 |
|                                                                      |
| Restatement principle: Restitutio in Integrum --- make the person    |
| whole, restore them to where they would have been.                   |
