**NEW B --- Undue Hardship (Two Legal Contexts)**

+----------------------------------------------------------------------+
| **CONTEXT 1 --- Human Rights / Accommodation (Ontario Human Rights   |
| Code, R.S.O. 1990, c. H.19)**                                        |
|                                                                      |
| Undue hardship is a legal standard --- not just a description of     |
| suffering.                                                           |
|                                                                      |
| Service providers must accommodate disability up to the point of     |
| undue hardship.                                                      |
|                                                                      |
| Test: cost, outside funding sources, health and safety.              |
|                                                                      |
| Source: OHRC Policy on Ableism and Discrimination Based on           |
| Disability --- ohrc.on.ca                                            |
|                                                                      |
| CONTEXT 2 --- Personal Injury / CMH Cascade                          |
|                                                                      |
| In tort, undue hardship describes the burden CMH placed on Matt by   |
| discharging him homeless, unsupported, post-surgical, TBI.           |
|                                                                      |
| Feeds into: aggravated damages, future care costs, loss of earning   |
| capacity.                                                            |
|                                                                      |
| Source: Clements v. Clements, 2012 SCC 32 --- but for the discharge, |
| downstream harms would not have occurred.                            |
