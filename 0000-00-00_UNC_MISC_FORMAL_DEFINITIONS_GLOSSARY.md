**FORMAL DEFINITIONS GLOSSARY**
===============================

**Key Terms as Defined in Ontario Social Assistance Law**

Programs: ODSP \| Ontario Works \| ADP

Purpose: Use these definitions when a caseworker uses a term that may
not mean what they imply it means.

Compiled: March 2026 \| Confidence flags applied to every entry

HOW TO USE THIS DOCUMENT: When a caseworker uses one of these terms
verbally or in writing, you can cite the legislative definition. If
their use of the term conflicts with the formal definition, document it
and raise it in any internal review or appeal. Caseworkers are bound by
the definition in the legislation and policy directives --- not their
own interpretation of common English.

**Person with a Disability**
----------------------------

+----------------------------------------------------------------------+
| Source: ODSP Act, 1997, s.4 \| ✓ CONFIRMED                           |
|                                                                      |
| **✓ CONFIRMED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| A person who:(a) has a substantial physical or mental impairment     |
| that is continuous or recurrent and expected to last one year or     |
| more;(b) the direct and cumulative effect of the impairment on the   |
| person\'s ability to attend to personal care, function in the        |
| community, and function in a workplace results in a substantial      |
| restriction in one or more of these activities of daily living;      |
| and(c) the impairment, its likely duration, and the restriction in   |
| activities of daily living have been verified by a person with       |
| prescribed qualifications.                                           |
|                                                                      |
| **STRATEGIC USE** All three conditions (a), (b), and (c) must be     |
| satisfied. \'Substantial\' and \'substantial restriction\' are key   |
| thresholds --- not total inability, but meaningful limitation. The   |
| cumulative effect of multiple impairments is assessed together, not  |
| each one in isolation.                                               |
+----------------------------------------------------------------------+

**Substantial Impairment**
--------------------------

+----------------------------------------------------------------------+
| Source: ODSP Act, 1997, s.4(1)(a) \| ✓ CONFIRMED -- appears in Act   |
| text; interpretation developed through SBT caselaw \| \~ RECALLED    |
| for SBT interpretations                                              |
|                                                                      |
| **✓ CONFIRMED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| Not defined by a single percentage or threshold in the Act.          |
| \'Substantial\' is a qualitative standard. The impairment must be    |
| continuous or recurrent and expected to last one year or more. Case  |
| law from the Social Benefits Tribunal has interpreted                |
| \'substantial\' as meaning more than trivial but not necessarily     |
| total.                                                               |
|                                                                      |
| **STRATEGIC USE** A caseworker who says your impairment is \'not     |
| substantial enough\' must be able to point to the standard they are  |
| applying. If they cannot define it, that is grounds for challenge in |
| internal review.                                                     |
+----------------------------------------------------------------------+

**Substantial Restriction in Activities of Daily Living**
---------------------------------------------------------

+----------------------------------------------------------------------+
| Source: ODSP Act, 1997, s.4(1)(b) \| ✓ CONFIRMED                     |
|                                                                      |
| **✓ CONFIRMED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| The direct AND cumulative effect of the impairment must result in a  |
| substantial restriction in one or more of: (1) personal care, (2)    |
| functioning in the community, or (3) functioning in a workplace.Only |
| ONE of the three domains needs to be substantially restricted.       |
|                                                                      |
| **STRATEGIC USE** Key word: \'cumulative.\' If you have multiple     |
| conditions, all of them together are assessed. A caseworker who      |
| evaluates each diagnosis separately and finds none \'substantial     |
| enough\' on its own is misapplying the law. Cite s.4(1)(b) directly. |
+----------------------------------------------------------------------+

**Benefit Unit**
----------------

+----------------------------------------------------------------------+
| Source: O. Reg. 222/98, s.1 \| ✓ CONFIRMED                           |
|                                                                      |
| **✓ CONFIRMED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| The recipient and any dependants included in the determination of    |
| income support. Benefits are calculated for the benefit unit as a    |
| whole, not per individual.                                           |
|                                                                      |
| **STRATEGIC USE** Income and assets of all benefit unit members are  |
| pooled in calculations. Knowing who is (and is not) included in your |
| benefit unit affects every financial calculation.                    |
+----------------------------------------------------------------------+

**Dependant**
-------------

+----------------------------------------------------------------------+
| Source: O. Reg. 222/98, s.1(1) \| ✓ CONFIRMED                        |
|                                                                      |
| **✓ CONFIRMED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| In relation to an applicant or recipient, means:(a) a person who     |
| resides in the same dwelling place and is: (i) the spouse of the     |
| applicant or recipient, (ii) a dependent child of the applicant or   |
| recipient or of their spouse, or (iii) a dependent adult of the      |
| applicant or recipient or of their spouse; OR(b) a spouse who is     |
| absent from the dwelling place if the absence is for a reason other  |
| than a breakdown in the relationship with no reasonable prospect of  |
| reconciliation.                                                      |
|                                                                      |
| **STRATEGIC USE** The definition of \'dependant\' matters when       |
| determining household income counts and benefit unit size. A         |
| caseworker who includes or excludes a household member incorrectly   |
| can miscalculate your entitlement.                                   |
+----------------------------------------------------------------------+

**Mandatory Benefit**
---------------------

+----------------------------------------------------------------------+
| Source: ODSP Act s.5; O. Reg. 222/98; ODSP Directives 9.12, 9.13 \|  |
| ✓ CONFIRMED                                                          |
|                                                                      |
| **✓ CONFIRMED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| A benefit that must be provided when the legislated or regulatory    |
| eligibility criteria are met. Staff have no discretion to deny a     |
| mandatory benefit to an eligible recipient. Examples confirmed: ADP  |
| co-payment (25%), mobility device battery/repair, MSN items verified |
| by approved health care professional.                                |
|                                                                      |
| **STRATEGIC USE** If a mandatory benefit is denied, the caseworker   |
| has made a legal error --- not an exercise of discretion. The appeal |
| path is straightforward: the question is not \'should we provide     |
| this\' but \'are the eligibility criteria met.\' Frame denials of    |
| mandatory benefits as errors of law, not policy disagreements.       |
+----------------------------------------------------------------------+

**Discretionary Benefit**
-------------------------

+----------------------------------------------------------------------+
| Source: OW Act 1997; OW Directive 7.4; ODSP preamble \| ✓ CONFIRMED  |
|                                                                      |
| **✓ CONFIRMED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| A benefit that an Administrator or Director may provide, based on    |
| individual circumstances, within available budget and policy         |
| parameters. The decision involves judgment. However, discretion must |
| be exercised in good faith, consistently, and within the purposes of |
| the legislation.OW Directive 7.4 confirmed: \'Ontario Works          |
| mandatory and discretionary employment benefits, as part of basic    |
| financial assistance, are provided to participants if the            |
| Administrator is satisfied that eligibility criteria have been       |
| met.\'                                                               |
|                                                                      |
| **STRATEGIC USE** Discretion is not unlimited. A refusal to even     |
| consider a request, or a blanket policy of denial without individual |
| assessment, is an improper exercise of discretion and is appealable. |
| The question on appeal is whether discretion was properly exercised, |
| not whether you agree with the outcome.                              |
+----------------------------------------------------------------------+

**Exceptional Circumstances**
-----------------------------

+----------------------------------------------------------------------+
| Source: ODSP Act, 1997, s.6; ODSP Directive 9.11 (Hearing Aids) \| ✓ |
| CONFIRMED for s.6; ✓ CONFIRMED for directive 9.11                    |
|                                                                      |
| **✓ CONFIRMED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| Two distinct uses:(1) ODSP Act s.6: Lieutenant Governor in Council   |
| may by order direct income support to a person not otherwise         |
| eligible, where investigation shows the advisability of providing    |
| support in exceptional circumstances.(2) Directive 9.11 (Hearing     |
| Aids) Exceptional Circumstances Policy: A request may be made for an |
| item/service not listed in the MCSS Schedule, or listed but          |
| requiring exceptional circumstances approval, where a medical        |
| indication exists. Decisions made by Social Assistance Centralized   |
| Services Branch (SACSB).                                             |
|                                                                      |
| **STRATEGIC USE** These are two separate mechanisms. The s.6         |
| mechanism is rarely used and involves cabinet-level orders. The      |
| Directive 9.11 Exceptional Circumstances Policy is a real            |
| administrative pathway for obtaining items not in the standard       |
| schedule when medical necessity is established.                      |
+----------------------------------------------------------------------+

**Mandatory Special Necessities (MSN)**
---------------------------------------

+----------------------------------------------------------------------+
| Source: ODSP Directive 9.12 \| ✓ CONFIRMED                           |
|                                                                      |
| **✓ CONFIRMED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| A mandatory benefit covering the costs of eligible medically         |
| necessary supplies, items, and devices required as a direct result   |
| of a surgical, radiological or medical procedure or disease, where   |
| not reimbursed from any other source.Key conditions confirmed:-      |
| Verified by an approved health care professional- Item must appear   |
| on the MSN Schedule of benefits- Approval only for entry-level       |
| devices or core models providing basic functionality- Based on most  |
| economical costs                                                     |
|                                                                      |
| **STRATEGIC USE** \'Entry-level\' and \'most economical\' are        |
| constraints, not absolute bars. If the entry-level device is         |
| clinically inadequate for your specific needs, document why with     |
| clinical support and request an exception or appeal. The word        |
| \'mandatory\' means denial requires legal justification, not         |
| caseworker preference.                                               |
+----------------------------------------------------------------------+

**Medically Necessary**
-----------------------

+----------------------------------------------------------------------+
| Source: \~ RECALLED -- not formally defined in ODSP Act or           |
| Regulations. Interpreted through SBT caselaw and clinical practice.  |
| ADP and MSN use \'medically necessary\' as a functional standard.    |
|                                                                      |
| **\~ RECALLED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| Not defined in statute. Used as a functional standard in directives: |
| the item or service must be clinically required to address a health  |
| need arising from disability or medical condition --- not merely     |
| convenient or beneficial.Practical application: Requires documented  |
| clinical justification from an approved health care professional.    |
| Generic references to a condition are insufficient; specific         |
| functional need linked to the item must be documented.               |
|                                                                      |
| **STRATEGIC USE** Because this term is not defined in legislation, a |
| caseworker\'s subjective determination that something is not         |
| \'medically necessary\' can be challenged. The standard should be    |
| based on clinical evidence, not administrative preference. An OT or  |
| physician recommendation carries significant weight.                 |
+----------------------------------------------------------------------+

**Reasonable (as used in policy)**
----------------------------------

+----------------------------------------------------------------------+
| Source: \~ RECALLED -- used throughout ODSP and OW regulation. O.    |
| Reg. 222/98 uses \'reasonable efforts\' and \'reasonable period.\'   |
| Not centrally defined.                                               |
|                                                                      |
| **\~ RECALLED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| Used in multiple contexts:- \'Reasonable efforts to sell\' real      |
| property (O. Reg. 222/98)- \'Reasonable period\' for application of  |
| funds toward exempt assets- \'Reasonably necessary\' for advance     |
| child care payments (OW Directive 7.4)- \'Most economical/reasonable |
| costs\' for MSN medical transportationIn each case, \'reasonable\'   |
| is assessed based on individual circumstances, not a fixed standard. |
|                                                                      |
| **STRATEGIC USE** When a caseworker says something is \'not          |
| reasonable,\' ask them to define the standard they are applying and  |
| document it. \'Reasonable\' is a fact-specific determination that    |
| can be reviewed on appeal.                                           |
+----------------------------------------------------------------------+

**Director (ODSP)**
-------------------

+----------------------------------------------------------------------+
| Source: ODSP Act, 1997, various \| ✓ CONFIRMED                       |
|                                                                      |
| **✓ CONFIRMED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| The Director of the Ontario Disability Support Program, appointed    |
| under the Act. Caseworkers exercise powers delegated from the        |
| Director. The Director has specific decision-making authority        |
| including over internal reviews. Under s.39 and 43 of the ODSP Act,  |
| delivery sites exercise powers and duties of the Director for        |
| financial determinations.                                            |
|                                                                      |
| **STRATEGIC USE** When a caseworker makes a decision, they are doing |
| so under delegated authority from the Director. This matters for     |
| appeals: the Tribunal cannot make a decision the Director has no     |
| authority to make (ODSP Act s.29(3)).                                |
+----------------------------------------------------------------------+

**Income Support**
------------------

+----------------------------------------------------------------------+
| Source: ODSP Act, 1997, s.3 \| ✓ CONFIRMED                           |
|                                                                      |
| **✓ CONFIRMED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| Shall be provided to a person with a disability (as determined under |
| s.4) and to a person of a prescribed class. Income support shall be  |
| provided for the benefit of the eligible person and their            |
| dependants.                                                          |
|                                                                      |
| **STRATEGIC USE** Income support is a right when eligibility         |
| criteria are met, not a grant of administrative discretion. The Act  |
| uses mandatory language (\'shall be provided\').                     |

**Benefit-Conferring Legislation (Interpretive Principle)**
-----------------------------------------------------------

+----------------------------------------------------------------------+
| Source: Rizzo v Rizzo Shoes \[1998\] 1 SCR 27, applied to ODSP by    |
| legal commentary \| \~ RECALLED                                      |
|                                                                      |
| **\~ RECALLED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| A principle of statutory interpretation. Legislation that confers    |
| benefits (such as the ODSP Act) must be interpreted so that any      |
| genuine ambiguity in the legislation is resolved in favour of the    |
| person claiming the benefit.Applied to ODSP: Where a policy          |
| directive or regulatory provision is ambiguous, the interpretation   |
| that provides the benefit should be preferred.                       |
|                                                                      |
| **STRATEGIC USE** This is a Supreme Court of Canada principle. It is |
| not simply an advocacy position --- it is law. Any caseworker or     |
| supervisor who applies a narrow interpretation of an ambiguous       |
| provision without justifying why that interpretation is the correct  |
| one is potentially misapplying the law. Cite this in writing in any  |
| internal review where an ambiguous provision has been interpreted    |
| against you.                                                         |

**Overpayment**
---------------

+----------------------------------------------------------------------+
| Source: ODSP Act, 1997, s.14; O. Reg. 222/98 \| ✓ CONFIRMED          |
|                                                                      |
| **✓ CONFIRMED**                                                      |
|                                                                      |
| **FORMAL DEFINITION**                                                |
|                                                                      |
| Income support paid to a person who was not entitled to it or who    |
| received more than the entitled amount. May be recovered by: (1)     |
| reduction of income support under s.15, (2) notice under s.16, (3)   |
| legal proceeding under s.17. Deduction from income support shall not |
| exceed the prescribed amount unless the recipient agrees to greater  |
| deductions.                                                          |
|                                                                      |
| **STRATEGIC USE** You have the right to know the exact amount        |
| claimed and the basis for the claim. The recovery rate from ongoing  |
| income support is capped. An overpayment determination is itself     |
| subject to internal review and appeal.                               |
